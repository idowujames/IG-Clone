## About
This repository demonstrates an Instagram-like layout (header, feed, stories, profile card) implemented with semantic HTML and plain CSS. Intended for learning and reference.

## Built with
- HTML5
- CSS3 (responsive layout and basic components)
- No frameworks or build tools required

## Features
- Responsive header and navigation
- Post/feed layout with image placeholders
- Profile summary / sidebar card
- Basic interactions styled with CSS (hover states)
